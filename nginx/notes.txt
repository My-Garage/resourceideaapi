When a new server / droplet is created, nginx is to be installed and secured with ssl to enable secure connections.

1. Make sure your ssh keys are added on digitalocean.
2. SSH to the server. 'ssh root@server_ip' Enter the passphrase for your private key.
3. Install nginx and enable security with ssh.
4. Restart nginx.
5. Try accessing the site in the browser and esnure it's resolving on https, with custom domain pointing to IP:Port
